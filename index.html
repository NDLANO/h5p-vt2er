H5P "Virtual Tour to Escape Room" converter

NDLA has created the content type Escape Room. It is based on H5P Group's Virtual Tour 360 content type, but with additional features. This tool allows you to convert Virtual Tour 360 contents to Escape Room content.

You can upload a Virtual Tour 360 content and will be offered to download the converted content.

Please note that you will need to have installed the Escape Room content type on your H5P platform before, or someone with administrative rights to your platform will need to upload the resulting file in order to install missing libraries.
